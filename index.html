Cyclistic Dataset

This SQL query was executed on Google BigQuery, resulting in the creation of a merged target table. This table combines the New York Citibike public datasets with the zip code table that was uploaded. The merge was performed using a JOIN operation, which matched the relevant data based on a common attribute, the zip code. The resulting merged table, which integrates information from both datasets, served as the foundation for constructing the Cyclistic dashboard.
